# 📊 EDA on Retail Sales Data

## 📌 Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on a retail sales dataset to identify customer purchasing patterns, sales trends, and business insights.

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook (VS Code)

## 📂 Dataset
Retail Sales Dataset containing transaction details, customer demographics, product categories, quantity, price per unit, and total sales.

## 📈 Analysis Performed
- Initial data inspection
- Missing value analysis
- Duplicate check
- Descriptive statistics
- Monthly sales trend
- Quarterly sales trend
- Customer age group analysis
- Gender distribution
- Revenue by product category
- Correlation heatmap
- Additional visualization
- Business recommendations

## 💡 Key Insights
- Sales vary across different months and quarters.
- Electronics have a higher average transaction value.
- Customer purchasing behavior differs across age groups.
- Product categories contribute differently to overall revenue.

## 📌 Business Recommendations
1. Increase inventory for high-performing product categories.
2. Run promotions during low-sales months.
3. Target marketing campaigns toward active customer age groups.
4. Use sales insights to improve inventory planning.

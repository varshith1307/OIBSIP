# Data Cleaning Project

## Objective
Clean a messy dataset by handling missing values, duplicates, inconsistent formatting, outliers, and incorrect data types.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Tasks Performed
- Data Quality Report
- Missing Value Handling
- Duplicate Removal
- Data Standardization
- Outlier Detection using IQR
- Data Type Correction
- Before vs After Comparison
- Export Cleaned Dataset

## Output
A cleaned Titanic dataset (`cleaned_titanic.csv`) ready for further analysis.
